# EE461L Frontend Technologies Homework

This repository contains two assignments for EE461L:

## Task 1: Personal Website (HTML/CSS)

A single-page personal website built with pure HTML and CSS, deployed via GitHub Pages.

- **Live site:** Enable GitHub Pages from Settings → Pages → Deploy from branch `main`, root `/`
- **Files:** `index.html`, `stylesheet.css`, `images/`

## Task 3: React Projects App

A custom React front end for a hardware checkout / project management interface.

- **Location:** `projects-app/`
- **See:** `projects-app/README.md` for install, run, and submission instructions

## GitHub Pages Deployment

1. Push all changes to the `main` branch
2. Go to your repo on GitHub → **Settings** → **Pages**
3. Under "Source", select **Deploy from a branch**
4. Choose branch **main** and folder **/ (root)**
5. Click **Save**
6. Wait 1–2 minutes, then visit `https://manas2006.github.io/software-frontend-hw/`
